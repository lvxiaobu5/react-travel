## 项目简介

携程旅游网项目包含从首页浏览商品列表，商品详情，注册登录，模糊搜索，加入购物车，提交订单，订单管理等功能。

当前仓库为携程旅游网的**项目模板**。

### 技术栈

- 前端框架：[React](React18 + TS + Setup)
- 状态管理：[react-redux]
- 组件库：[Ant Design]

## 运行程序

1. 安装依赖

```shell
# npm
npm i --registry=https://registry.npmmirror.com

# pnpm
pnpm i --registry=https://registry.npmmirror.com
```

2. 运行程序

```shell
npm run dev
```

## 打包程序
```shell
npm run build
```
